# Boot files

[![Build](https://img.shields.io/github/actions/workflow/status/ipxe/boot.ipxe.org/build.yml)](https://github.com/ipxe/boot.ipxe.org/actions/workflows/build.yml?query=branch%3Amaster)

This repository is responsible for building the https://boot.ipxe.org
static boot files site for [iPXE](https://ipxe.org), including the
rolling release binaries built from the most recent commit on the
[iPXE `master` branch](https://github.com/ipxe/ipxe).
